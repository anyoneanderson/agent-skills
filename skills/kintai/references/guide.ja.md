# kintai リファレンスガイド（日本語）

## 環境設定

### .env ファイル

プロジェクトルートまたはホームディレクトリに `.env` ファイルを作成:

```
LEVTECH_EMAIL=your-email@example.com
LEVTECH_PASSWORD=your-password
```

**注意**: `.env` ファイルは `.gitignore` に追加し、リポジトリにコミットしないこと。

### Cookie 保存先

- パス: `/tmp/kintai-cookies.json`
- 形式: ブラウザ自動操作ツールの Cookie JSON 形式
- OS 再起動時にクリアされる（再ログインが必要）

## 認証フロー詳細

### Google ログイン手順

```
1. https://platform.levtech.jp にアクセス
   - 待機: ページのロード完了を確認

2. ログインページが表示された場合:
   a. 「Googleでログイン」ボタンを探してクリック
      - セレクタ候補: テキスト "Google" を含むボタン/リンク
   b. Google ログインページで:
      - メールアドレス入力フィールドにメールを入力
      - 「次へ」ボタンをクリック
      - 待機: パスワード入力画面のロード
      - パスワード入力フィールドにパスワードを入力
      - 「次へ」ボタンをクリック
   c. リダイレクト待機:
      - URL が platform.levtech.jp に戻るまで待機
      - ダッシュボードまたは作業報告ページの表示を確認

3. ログイン成功確認:
   - ページ内にユーザー名が表示されていることを確認
   - Cookie を /tmp/kintai-cookies.json に保存
```

### Cookie 再利用

```
1. /tmp/kintai-cookies.json が存在するか確認
2. 存在する場合:
   a. Cookie をブラウザセッションに読み込み
   b. https://platform.levtech.jp/p/workreport/ にアクセス
   c. ログインページにリダイレクトされたら → Cookie 無効 → 再ログイン
   d. 作業報告ページが表示されたら → Cookie 有効 → 続行
3. 存在しない場合:
   → Google ログイン手順を実行
```

## ブラウザ操作詳細

### 作業報告一覧ページ → 詳細ページ

```
URL: https://platform.levtech.jp/p/workreport/

1. ページ内のテーブルから当月のリンクを探す
   - テキスト例: "2026/03"
   - リンクをクリック

2. 詳細ページに遷移
   - URL パターン: /p/workreport/input/{id}/
   - ページタイトルに「作業報告書詳細」を含む
```

### 詳細ページ → 編集モード

```
1. 「編集する」ボタンを探してクリック
   - テキスト: "編集する"
   - 待機: 入力フィールドが表示されるまで

2. 編集モードの確認:
   - 「保存する」ボタンが表示されている
   - 日付行に input フィールドが表示されている
```

### 日付行の特定と入力

```
1. テーブルの各行を走査
   - 各行の最初のセルに日付テキスト（例: "03/04"、"03/04（水）"）
   - 対象日付のテキストを含む行を特定

2. 入力フィールドの特定:
   - 行内の input 要素を順番に取得
   - 1番目: 開始時刻（例: "10:00"）
   - 2番目: 終了時刻（例: "19:00"）
   - 3番目: 休憩時間（例: "01:00"）

3. 値の入力:
   - 各フィールドをクリックしてフォーカス
   - 既存値をクリアしてから新しい値を入力
   - Tab キーで次のフィールドに移動（必要に応じて）

4. 保存:
   - 「保存する」ボタンをクリック
   - 待機: ページがリロードまたは更新されるまで
   - スクリーンショットを撮影して表示
```

### check モードの操作

```
1. 詳細ページ（閲覧モード）のテーブルを読み取り
   - 「編集する」は押さない
2. 各行の「開始」列（2列目）を確認
   - 値がある行 → 入力済み
   - 空の行 → 未入力
3. 入力済みの日付を収集
4. テキストで結果を表示:
   "入力済み: 3/3(火), 3/5(木), 3/7(土) — 3日"
```

## トラブルシューティング

### ログインできない

- `.env` のメールアドレスとパスワードを確認
- Google アカウントで 2FA が有効になっていないか確認（2FA は非対応）
- Cookie ファイルを削除して再試行: `rm /tmp/kintai-cookies.json`

### 当月のリンクが見つからない

- 作業報告一覧ページの URL が正しいか確認
- 当月の作業報告書が作成済みか確認（レバテック管理画面で確認）

### 日付行が見つからない

- 対象日付が当月内かどうか確認
- 日付フォーマットが正しいか確認（M/D 形式: 3/4, 12/25 等）

### 保存に失敗する

- 編集モードに入れているか確認（「保存する」ボタンが見えるか）
- 入力値のフォーマットが正しいか確認（HH:MM 形式）
