# エージェントプロファイル

## 対応エージェント

### Claude Code

| プロパティ | 値 |
|----------|-------|
| コマンド | `claude` |
| 自動承認 | `claude --dangerously-skip-permissions` |
| プロンプトパターン | 行頭の `>` またはセッション情報後の入力カーソル |
| 終了 | `/exit` または Ctrl+C |
| 起動時間 | 2〜5秒 |

### Codex (OpenAI)

| プロパティ | 値 |
|----------|-------|
| コマンド | `codex` |
| 自動承認 | `codex --dangerously-bypass-approvals-and-sandbox` |
| プロンプトパターン | カーソル付きプロンプト表示 |
| 終了 | `exit` または Ctrl+C |
| 起動時間 | 2〜5秒 |

### Gemini CLI (Google)

| プロパティ | 値 |
|----------|-------|
| コマンド | `gemini` |
| 自動承認 | 非対応 — Gemini CLI には権限スキップオプションがなく、常にインタラクティブモードで動作 |
| プロンプトパターン | `>` またはプロンプトインジケータ |
| 終了 | `/exit` または Ctrl+C |
| 起動時間 | 2〜5秒 |

## カスタムエージェントの追加

上記以外のエージェントに委任するには、以下を指定してください:

1. **起動コマンド**: エージェントを起動するターミナルコマンド
2. **プロンプトパターン**: 入力待ち状態で表示されるもの
3. **終了コマンド**: エージェントを正常に終了する方法

cmux-delegate への指示例:

```
「'my-agent --interactive' を新しいワークスペースで起動して、このタスクを送って: ...」
```

親エージェント（Claude Code）がカスタムエージェントの準備完了・タスク完了を判断します。

## エージェント選択ガイドライン

| ユースケース | 推奨エージェント | 理由 |
|-------------|----------------|------|
| 一般的なコーディングタスク | Claude Code | フルコンテキスト、最高のコーディング能力 |
| コードレビュー / セカンドオピニオン | Codex | 異なる視点、レビューに向いている |
| クイックリサーチ | Gemini CLI | 高速なレスポンス |
| 自動承認タスク | Claude Code (auto) or Codex (auto) | 権限プロンプトなし |

## セキュリティ上の注意

- **自動承認モード**（`--dangerously-skip-permissions`, `--dangerously-bypass-approvals-and-sandbox`）は安全チェックをバイパスします。タスクと環境を信頼できる場合のみ使用してください。
- 親エージェントは子エージェントに任意のテキストを送信します。タスクプロンプトに機密データが含まれないよう注意してください。
